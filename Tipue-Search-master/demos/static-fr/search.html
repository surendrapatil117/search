?

Aide pour recherche

Vous pouvez ajouter :

  • Le caractère de troncature point d’interrogation (?) n'importe où dans un mot pour remplacer n'importe quel caractère unique.
    Exemple : étai? correspondra à étais et était.
  • Le caractère de troncature asterisk (*) au début ou à la fin d’un mot pour remplacer zéro ou plusieurs caractères.
    Exemple : étai* correspondra à était et étaient.
  • Des "guillemets droits double" autour des groupes de mots à rechercher.
    Exemple : "moteur de recherche".
  • Un –trait d'union avant un mot pour exclure de tous les résultats qui incluent ce mot. Pour les groupes de mots, mettre le "-trait d’union" à l'intérieur du guillemet droit doubel gauche pour exclure le groupe de mots.
    Exemple : -plugin ou "-jQuery plugin" exclura les résultats contenant le mot plugin ou le group des mots jQuery plugin.

Toutes les recherches sont insensible à la casse des caractères. Toutefois les caractères avec accents sont considérés différents que ceux sans accents. Exemple : Si vous cherchez le mot trouve, le moteur de recherche ne trouvera pas trouvé.
